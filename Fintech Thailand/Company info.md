# สรุปบริษัท: FINTECH (THAILAND) Co., Ltd.

## สถานะปัจจุบัน

* บริษัท fintech ขนาดเล็ก (micro infrastructure company)
* รายได้ต่ำกว่า 5 ล้านบาท/ปี
* ธุรกิจยังอยู่ในช่วง “ประคอง” มากกว่าเติบโต
* มี recurring usage จากลูกค้าเดิม แต่ยังไม่มี recurring revenue model ที่แข็งแรง
* รายได้หลักมาจาก:
  * ส่วนแบ่งค่าคอมมิชชั่นซื้อขายหุ้น
  * พัฒนาระบบ automate trading และระบบเกี่ยวกับการบริหารพอร์ตลงทุน

---

# ลูกค้า

ลูกค้าหลัก:

* Broker
* บริษัทจัดการ/บริหารพอร์ต
* นักลงทุนรายใหญ่

* ใช้งานต่อเนื่อง
* อยู่ใน ecosystem ตลาดทุนไทย
* เป็นตลาดเฉพาะทาง (niche market)

# จุดแข็งของบริษัท

## 1. Domain expertise ตลาดทุนไทย

* เข้าใจ workflow การลงทุนจริง
* เข้าใจ trading automation
* เข้าใจ broker ecosystem
* มีประสบการณ์เกี่ยวกับ Settrade OpenAPI SDK

---

## 2. Trust & Connection

* งานส่วนใหญ่มาจาก network และ shareholder
* มี credibility ในวงการเฉพาะทาง

---

## 3. ทีม lean และ technical

ทีม 4 คน:

* MD (พื้นฐาน Software Developer)
* Software Developer ที่มีความรู้การเงิน/การลงทุน
* UX/UI Designer
* Accountant

ข้อดี:

* burn ต่ำ
* ตัดสินใจเร็ว
* technical capability ดี

# จุดอ่อนหลัก

## 1. พึ่งพา network สูง

* ยังไม่มี distribution channel ของตัวเอง
* ไม่มี sales engine
* ถ้าไม่มีงานใหม่ บริษัทโตไม่ได้

---

## 2. Productization ยังไม่ชัด

ปัจจุบัน:

* มีแนวโน้มสร้าง product จาก intuition
* บาง product ถูกสร้างแต่ไม่มี adoption

ปัญหาหลัก:

* build ก่อน validate
* ยังไม่มี repeatable business model

---

## 3. ยังไม่มี recurring revenue machine ที่แข็งแรง

รายได้ยังมีลักษณะ:

* relationship-driven
* project-driven

ไม่ใช่:

* predictable
* scalable
* subscription-based
